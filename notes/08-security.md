
# Microsoft Defender for Cloud

## Objective

Explore Azure's cloud security platform to evaluate the security posture of cloud resources, review recommendations, and understand secure configuration best practices.

## Microsoft Defender for Cloud

Microsoft Defender for Cloud is Azure's cloud-native application protection platform (CNAPP). It provides security posture management, threat protection, and recommendations for improving the security of Azure resources.

## Features Reviewed

- Defender Overview
- Environment Settings
- Secure Score
- Security Recommendations
- Regulatory Compliance (if available)

## Secure Score

Secure Score provides a measurement of how well Azure resources follow Microsoft's recommended security best practices.

Improving the Secure Score reduces security risks by implementing recommended configurations.

## Security Recommendations

Recommendations help identify security improvements such as:

- Enable additional security features
- Improve identity protection
- Configure monitoring
- Strengthen network security
- Protect storage resources

## Environment Settings

Environment Settings determine which Defender plans and monitoring features are enabled for a subscription.

Organizations can customize security coverage based on workloads and compliance requirements.
