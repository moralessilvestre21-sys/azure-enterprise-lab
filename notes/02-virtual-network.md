# Virtual Network

## Purpose

A Virtual Network (VNet) provides a private network within Microsoft Azure that allows Azure resources to communicate securely.

## Configuration

- Name: vnet-enterprise-lab
- Region: East US
- Address Space: 10.0.0.0/16

## Why This Matters

The Virtual Network serves as the foundation of the cloud environment. It allows administrators to segment resources, control network traffic, and apply security policies.
