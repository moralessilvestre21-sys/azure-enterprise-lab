
# Microsoft Entra ID

## Purpose

Microsoft Entra ID is Microsoft's cloud-based identity and access management (IAM) service. It manages users, groups, authentication, and authorization for cloud resources.

## Lab Objectives

- Create cloud identities
- Organize users into security groups

- Prepare for Role-Based Access Control (RBAC)
